论文:Indoor Human Localization with Orientation using WiFi Fingerprinting
第一,人的朝向对ap值影响非常大,或者说人体的阻挡影响非常大(15dbm),具体到我们的研究而言,人多起来的时候情况就变得非常复杂了.
第二,该论文提到的定位精度问题,在我们的研究中似乎不存在?定位精度应该由其他组来确定,我们提供指纹库.
第三,该论文对于其他影响因素如手机型号,手握手机,环境大小,场景内人数等都没有讨论,未有太多参考价值.
#晚上
重点:
Estimating the number of people using existing WiFi access point
第一,论文中提出的剔除异常值的方法可以采用一下,以代替现在的直接取平均值方法.
第二,论文中接受设备固定(电脑,电视等),而我们的方法中采用手机,多了手等影响因素.
第三,论文中采用的是'单AP多接收设备'的方法.
第四,人数和方差之间的关系,使用的也不是值的衰减,而是在一段连续时间内方差的变化.
第五,rssi和人数关系不是线性的.论文中采用的支持向量机算法准确度较高.
第六,对于有/无估计和拥塞程度的估计,基于SVR的方法的准确率非常高.对于人数的估计,基于SVR的方法的准确率为0.772,远远低于有/无估计的准确率.
第七,随着人数的增加,误差增加.多径衰落.RSSI的方差变得更高,并且估计误差也变得更高.去除变化过高的异常值可以减少多径衰落的影响.
总结:该论文设置的实验环境相对我们的研究较为简单,但研究方向一致,非常有参考价值.
